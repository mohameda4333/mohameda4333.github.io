Freedom Project

Context

I am a student at HSTAT in the Software Engineering Program. The "Freedom Project" for SEP11 is a year-long project all about making something using JavaScript along with a third-party JS tool.

Content

For my project, I chose to independently study Kaboomjs in order to help me make a mini practice shooting game. Kaboomjs is a third-party JS tool that makes it easier for the user to code as it provides more functions and loops that make it easier to build sprites and control them. This is a promising tool to use, especially when making a game.

Reflection

This project was the result of everything I have learned in coding up until now. It had some connections to computer science specifically for me since I had to make sprites and code how they function. My tool was easy for making sprites so it was simpler to make the game that I wanted. Over this past year, I had many challenges that caused me to learn how teach myself and collaborate with others to help me solve the problem. I didn't know how to use functions and loops in kaboom so I asked my cousin who is a sophomore in college who majors in computer science to help me with my project. He helped me understand how to order the function correctly so that the sprite can act upon it's function. These skills are very useful in coding as it helped me work by myself and with other people to achieve my goal. My next steps is to make my game more advanced by making it more challenging which requires more learning.
